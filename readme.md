# üé¨ Movie Recommendation System

## üåü Overview
This project implements a content-based movie recommendation system that suggests similar movies to a user based on a selected title. The recommendations are generated by analyzing movie metadata, including genres, keywords, cast, and crew.

The core of the system relies on pre-computed similarity scores (likely Cosine Similarity) to ensure fast and efficient lookups.

## ‚ú® Features
* **Content-Based Filtering:** Recommends movies based on the characteristics of a chosen movie.
* **Fast Lookups:** Uses pre-calculated similarity data (`similarity.pkl`) for near-instantaneous results.
* **Data Processing:** Utilizes key metadata from the TMDB datasets for feature engineering.

## üíæ Data Source
The system is built using the following datasets from The Movie Database (TMDB), which are included in this repository:
* `tmdb_5000_movies.csv`: The primary dataset containing movie metadata such as title, genres, keywords, and overview.
* `tmdb_5000_credits.csv`: Contains the cast and crew information for the movies, used for feature extraction.
