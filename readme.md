Project Overview
This project features an interactive data visualization dashboard built using Power BI. The goal is to perform a comprehensive analysis of the Amazon Prime Video content library, providing key insights into content distribution, performance metrics, and audience trends.
The attached Amazon prime Dashboard.pbix file is the complete, interactive report.
üéØ Goals and Key Insights
The dashboard is designed to answer critical questions about the Amazon Prime content catalog:
1.	Content Mix: Analyze the breakdown of Movies vs. TV Shows and their growth over time.
2.	Genre Popularity: Identify the top-performing and most frequently added genres.
3.	Regional Focus: Visualize content distribution across various production countries.
4.	Rating Distribution: Understand how content is rated and the volume of content across different rating categories (e.g., TV-MA, PG-13).

üìà Dashboard Features
The report is structured into key pages, each providing a focused view:
1.	Home / Executive Summary: High-level KPIs including total number of titles, overall average rating, and key filters (Year, Type).
2.	Content Deep Dive: Detailed breakdown of genres, directors, and main cast members.
3.	Geographic Analysis: Map visualization showing content contribution by country of origin.
4.	Time Series Analysis: Trend lines for content additions per month and year, revealing platform growth patterns.
